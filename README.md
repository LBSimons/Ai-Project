# Ai-Project
ik ga met het gebruik van een ai model naar keuze (chatgpt) om een dating sim style spel te maken
waar je in principe een verhaal zit te volgen waar je naar leest en het verhaal probeert te volgen.
er zullen verschillende assets gemaakt worden door de ai zelf sinds dat hoe het project gedaan moet worden gemaakt
verder dan dat begeleid ik het ai model om op het rechte pad te blijven en het project te creeeren.
stap voor stap maken we het project zodat de ai niet in de war komt te krijgen
daarnaast blijven we in een gezamelijke chat omdat dat hoort tenzij het gesprek te ver af komt
voor het project ga ik unity gebruiken
